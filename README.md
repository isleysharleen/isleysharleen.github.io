# Mire Sirens + FENtastical for Kodi

A Kodi repository hosting the **Mire Sirens** video add-on along with the
**FENtastical** skin and its companion helper script.

## What's inside

|Add-on|Purpose|
|-|-|
|Mire Sirens|Video add-on (community continuation of the Fen Light family)|
|FENtastical|Custom skin (FENtastic fork, Estuary MOD)|
|FENtastical Helper|Required companion script for the FENtastical skin|

## Install

### Step 1 — Enable unknown sources (first time only)

If this is your first third-party Kodi add-on:

**Settings → System → Add-ons → Unknown sources → On**

(Kodi will warn you. That's expected — say yes.)

### Step 2 — Add the repository source

1. In Kodi: **Settings → File Manager → Add source**
2. Paste this URL: https://isleysharleen.github.io/packages/
3. Name the source: mire
4. Click **OK**

### Step 3 — Install the repository

1. **Settings → Add-ons → Install from zip file**
2. Choose **mire** (the source you just added)
3. Open **repository.miresirens**
4. Install **repository.miresirens-1.1.0.zip**
5. Wait for the "Mire Sirens Repository — Add-on installed" notification

### Step 4 — Install the add-ons

**Settings → Add-ons → Install from repository → Mire Sirens Repository**

Install in this order:

1. **Program add-ons → FENtastical Helper** (install first — the skin
depends on it)
2. **Look and feel → Skin → FENtastical**
3. **Video add-ons → Mire Sirens**

### Step 5 — Switch to the FENtastical skin

**Settings → Interface → Skin → FENtastical**

Kodi will ask if you want to keep the new skin — say yes.

## Updating

The repository auto-updates whenever a new version is published. To force
an update check:

**Settings → Add-ons → Check for updates**
